# Virtual Bank
This is a bank simulator. It can make a transfer between two accounts, can withdraw and deposit money.

To start the bank, run the following command in the console.
```
php index.php
```

### Output
```
John has: 400
Smith has: 300
---------------------------------------------------
Transfer 200 from John to Smith: 
John has now: 200
Smith has now: 500
---------------------------------------------------
Smith withdraws 400: 
John has now: 200
Smith has now: 100
---------------------------------------------------
John deposits 800: 
John has now: 1000
Smith has now: 100
```
